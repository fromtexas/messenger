# rails-messenger

Rails messenger with jquery and redis -> [Demo](http://test.crimsonraven.lclients.ru)

![rails-messenger](https://image.ibb.co/gvYb5m/rails_mess.png) <!-- .element width="400px" -->

## Includes
* Ruby version 2.4.0
* Rails 5.0.2
* jquery
* redis
* PG
* devise
* jquery-ui

## Features
* registration
* login 
* add users by email to contacts
* messaging
